## ⚡ Quick Overview

**SplitBill** adalah aplikasi untuk berbagi tagihan dengan teman. Terdiri dari:
- **Backend**: FastAPI (Python) - Port 8001
- **Frontend**: React Native + Expo (TypeScript) - Port 3000
- **Database**: MongoDB - Port 27017
