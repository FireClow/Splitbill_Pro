# Splitbill Pro

A simple command-line tool to split bills and expenses among a group of people.

## Features

- Add participants and expenses
- Automatically calculate each person's share
- Track who owes whom
- Support for unequal splits

## Requirements

- Python 3.8+

## Installation

```bash
git clone https://github.com/FireClow/Splitbill_Pro.git
cd Splitbill_Pro
pip install -r requirements.txt
```

## Usage

```bash
python splitbill.py
```

Follow the interactive prompts to add participants and expenses.

## Running Tests

```bash
python -m pytest tests/
```
